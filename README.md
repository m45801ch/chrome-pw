# 瀏覽器密碼鎖定助手 🔒

一個 Chrome 擴充套件，在啟動瀏覽器或閒置時以全螢幕鎖定畫面保護您的瀏覽器隱私，防止他人未經授權使用您的 Chrome。

## 功能特性

- **全螢幕鎖定防護** — 鎖定後強制顯示鎖定畫面，自動最小化其他視窗、攔截新分頁
- **多種登入時效模式**：
  - 每次重開瀏覽器
  - 當日有效（至 23:59）
  - 每日指定時間點
  - 自訂倒數計時（天/時/分）
- **安全密語救援** — 首次設定時強制填寫，鎖定畫面輸入密語即可解密顯示原始密碼
- **Email 救援**（選用） — 透過 Google Apps Script 發送驗證碼，可重設密碼
- **自動重設安全閥** — 鎖定超過指定天數（預設 3 天）自動移除鎖定，避免永久鎖死
- **特定時段強制鎖定** — 設定時間區間，時間到自動上鎖
- **一鍵鎖定** — 點擊工具列圖示即可立即鎖定

## 安裝方式

1. 下載此專案的所有檔案
2. 開啟 Chrome，前往 `chrome://extensions`
3. 開啟右上角「開發人員模式」
4. 點擊「載入未封裝項目」，選取專案資料夾
5. 安裝完成後，點擊工具列圖示即可開始使用

## 使用說明

### 首次設定
安裝後會自動開啟設定畫面，需設定：
- **解鎖密碼** — 用來解鎖瀏覽器
- **安全密語** — 必填，忘記密碼時可輸入此密語還原密碼
- **救援 Email**（選填） — 用於 Email 救援

### 鎖定與解鎖
- **自動鎖定**：依設定的時效模式自動觸發
- **手動鎖定**：點擊工具列圖示 → 「立即鎖定瀏覽器」
- **解鎖**：在鎖定畫面輸入密碼即可

### 偏好設定
點擊工具列圖示 → 「偏好設定」（需先輸入密碼驗證），可調整：
- 登入有效時效
- 變更密碼與密語
- Email 救援配置
- 自動重設天數
- 特定時段鎖定

## 救援機制

| 方式 | 需事先設定 | 說明 |
|------|-----------|------|
| 安全密語 | 首次設定時強制填寫 | 輸入密語後解密顯示原始密碼 |
| Email 救援 | 需填 Email + 部署 GAS | 發送 6 位數驗證碼至信箱，可重設密碼 |
| 自動重設 | 預設啟用（3 天） | 鎖定超過天數自動移除鎖定 |

## Email 救援部署（選用）

若需要使用 Email 救援功能，請自行部署 Google Apps Script：

1. 前往 [script.google.com](https://script.google.com) 建立新專案
2. 貼入 `gas_template.txt` 的程式碼
3. 部署為「網頁應用程式」→ 執行身份設為「自己」→ 開放任何人存取
4. 將產生的網址填入設定的「GAS Web App URL」欄位

## 技術架構

- Manifest V3
- Web Crypto API（SHA-256 / AES-GCM）
- Service Worker
- Chrome Extension APIs（windows, tabs, storage, alarms）
